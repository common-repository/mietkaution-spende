== Description ==

Der Deutsche Mietkautionsbund e.V. veranstaltet regelmäßig einen Spendenmarathon, der das Ziel hat möglichst viel Geld für karitative Zwecke zu sammeln.

Dieses Plugin dient dazu, mit Hilfe von linkwash, eine einfachste Integration für den jeweiligen Webseitenbetreiber (Blogger) zu gewährleisten.

Nach der Integration des Plugins ist keinerlei Registrierung, weder beim Deutschen Mietkautionsbund e.V, noch bei linkwash nötig. Selbst bei der bloßen Integration des Plugins und der Sichtbarkeit auf der Webseite, spendet der Mietkautionsbund bereits einige Euro an eine karitative Einrichtung.

Das Plugin ermöglicht es weiterhin, nach Nachfrage des Administrators den aktuellen Spendenstand auf der eigenen Webseite anzuzeigen. Für jedes gewonnene Mitglied und jede beantrage Mietkaution werden ebenfalls einige Euro gespendet.

Die letzte Aktion dieser Art, trug den Namen "Mietkaution mit Herz" und dabei kamen insgesamt 717 Euro an Spendengeldern zusammen.

== Installation ==

1. Laden Sie den Mietkaution-Spenden Ordner in das Verzeichnis /wp-content/plugins/ hoch.

2. Aktivieren Sie das Mietkaution-Spenden Plugin im „Plugins“-Menü in Wordpress.
